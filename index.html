1. Information Collection and Use

No automatic data collection: The Application is designed not to collect, log, or transmit any personally identifiable information unless a specific optional feature (such as cloud backup or sync) is explicitly enabled by you. In the baseline configuration, the Application does not request your name, email address, location, contacts, or device identifiers.

If the Application requires internet connectivity for certain features (e.g., fetching public updates or displaying online content), standard technical data such as ephemeral IP addresses may be temporarily used to facilitate network communication. However, the Service Provider does not retain, monitor, or store this data in logs nor link it to any persistent identifier. No analytics, fingerprinting, or tracking mechanisms are employed.

Local storage only: Any content you generate (documents, media, game progress, preferences, or entries) is saved exclusively on your device's internal storage or external storage (if you grant storage permission). The Application does not automatically upload, sync, or share your local files with any remote server unless you explicitly initiate a backup or export function that you control.

2. App Permissions and Their Use

The Application may request certain Android permissions to deliver core features. Permissions are never used to collect personal data without your knowledge. Below is a generic description of possible permissions and how they are used responsibly:

  • Internet permission: Used only for features that require connectivity (e.g., downloading updates, fetching license validation, or accessing online help). No personal information is transmitted to any server unless you are using an explicit online feature (such as a webview or API call that you trigger).
  • Storage permission (read/write external storage): Used to save or access user-generated files (e.g., exporting reports, saving photos, or loading documents). All operations are user-initiated, and files remain under your control.
  • Camera or microphone permission: Used only for specific functionality such as scanning a QR code, capturing an image for profile purposes, or recording audio within the app session. No media is transmitted off-device without your explicit action.
  • Location permission (fine/coarse): If present, location is used exclusively for features that require geolocation (e.g., local weather, maps, or location-based reminders). Location data is processed on-device and never stored or shared with the Service Provider or third parties unless you choose to share it.

Permissions are requested at runtime (Android 6.0+) and you may deny or revoke any permission without affecting the core functionality unless the permission is essential for the feature.

3. Third-Party Access and SDKs

By default, this Application contains no third-party analytics, advertising, or crash reporting SDKs unless explicitly described in the app's store listing. We prioritize privacy by avoiding trackers. In the case where an app version includes optional third-party libraries (e.g., for ad-supported free versions or Firebase Cloud Messaging), we will:

  • List all third-party SDKs within the app's description or within this policy.
  • Ensure that such SDKs comply with data protection laws and only collect minimal necessary data with your consent.
  • Provide a clear opt-out or consent mechanism before any data collection begins.

If you are using a version of the Application that includes no external SDKs, rest assured that no third parties receive any information from the Application. The Service Provider does not sell, rent, or trade any user information.

Note for developers: If your app uses any third-party services (Google Analytics, AdMob, Facebook SDK, Crashlytics, etc.), you must list them here and update the policy accordingly. This generic template assumes zero third-party data sharing unless otherwise stated.

4. Location Information

The Application does not collect real-time location information unless a specific feature requires it and you provide explicit permission. If location access is requested, it is used solely for the feature you activate (e.g., displaying nearby points of interest). Location data is never stored persistently or transmitted to the Service Provider's servers. You can disable location permissions at any time via Android settings.

5. Your Opt-Out Rights and Device Control

Because the Application collects no personal information during normal usage, you do not need to opt out of data collection. If you wish to remove the Application entirely, uninstall it using the standard Android uninstallation process. Uninstalling will delete all locally stored data associated with the app (documents, settings, progress). No residual data remains on our side.

Voluntary communications: If you contact us via email or support form and provide personal information (e.g., name or email address), you may request deletion of that correspondence at any time by emailing our contact address. We will honor deletion requests promptly.

6. Children's Privacy

The Application is not directed to children under the age of 13 (or the applicable age of consent in your region). The Service Provider does not knowingly collect personal information from children. Since the Application's default behavior collects no data whatsoever, there is no risk of accidental data collection. If you believe a child has provided us with personal information through unofficial channels (e.g., direct email), please contact us, and we will delete that information.

7. Data Security

We take security seriously. Because the Application does not transmit your personal data over networks, the primary security risk lies in device-level access. We recommend that you keep your Android device updated, use screen locks, and avoid rooting your device if you store sensitive information. For any voluntary correspondence, we implement commercially reasonable measures to protect against unauthorized access, disclosure, or alteration.

Notifications will be made via email or a prominent notice on our official website.
